Desktop Apps
🖥 Run Queries UI ›

A local desktop app for running queries. Opens in your browser — no internet required after setup. Mac only.

⬇ Download ZIP
1
Download and unzip
Download the ZIP above and unzip it to a permanent location (e.g. ~/Documents/runqueries_ui). Don't leave it in Downloads.
2
Double-click launch.command
Inside the unzipped folder, double-click launch.command. On first run macOS may block it — right-click → Open to allow it.
3
Wait for first-time setup
The first launch installs dependencies automatically. A browser window will open at http://localhost:8501 when ready.
4
To quit
Close the Terminal window that opened, or press Ctrl+C in it.
Requires Python 3 — install from python.org if not already installed. Re-download and replace the folder to update.
Chrome Extensions
📖 How to install a Chrome extension ›
1
Open Chrome Extensions
Navigate to chrome://extensions in your browser address bar.
2
Enable Developer Mode
Toggle Developer Mode on in the top-right corner of the extensions page.
3
Click "Load unpacked"
Press the Load unpacked button that appears once Developer Mode is on.
4
Select the unzipped folder
Download the ZIP from this page, unzip it, then select the unzipped folder. Make sure to save it somewhere permanent — not in Downloads where it might get cleared.
Note: Extensions loaded this way are not auto-updated. Re-download and reload whenever a new version is published here.
